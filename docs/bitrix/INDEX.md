# Документация для бэкенда (Bitrix) — оглавление

Все инструкции по подключению нашей вёрстки к Bitrix. Начни с пунктов 1–2, дальше — по задаче.

## 1. Сборка и выкладка (обязательно сначала)
- **[UPDATE.md](UPDATE.md)** — что и как обновить: сборка с `BASE=/bitrix/templates/auth/`,
  выкладка `dist` в корень шаблона, чек-лист. Закрывает шрифты, ленивые чанки, лоадеры, фиксы.
- **[assets.php](assets.php)** — подключение собранных CSS/JS по `manifest.json`
  (вход ищется по `name === 'app'`, `$dir = SITE_TEMPLATE_PATH`). Заменяет ваш старый `assets.php`.

## 2. Таблицы (Сотрудники / Аптеки / Менеджеры)
- **[TABLES.md](TABLES.md)** — главный гайд: бэк отдаёт один JSON, фронт строит таблицу,
  сортировку, фильтры, пагинацию. Контракт колонок, типы ячеек, заголовок + кнопка «Экспорт».
- **[datatable-config.php](datatable-config.php)** — PHP-пример отдачи JSON-конфига таблицы.
- **[datatable-endpoint.php](datatable-endpoint.php)** — *запасной* серверный режим
  (`mode:"server"`, постраничная подгрузка) — нужен только при десятках тысяч строк.
- **[README.md](README.md)** — расширенный разбор таблиц/фильтров (серверный режим,
  доставка конфига файлом, AJAX). Справочно; для типового подключения хватает `TABLES.md`.

## 3. Задачи (раздел «Задачи» / на странице сотрудника)
- **[TASKS.md](TASKS.md)** — список задач из JSON: контракт задачи (состояния:
  активна / скрыта / выполнена / ждёт подтверждения / проверена), подключение через `data-tasks-src`.
- **[tasks-config.php](tasks-config.php)** — PHP-пример отдачи JSON со списком задач.

## Общие правила (везде одинаково)
- Сборка только под **node 22**, с **`BASE=/bitrix/templates/auth/`**.
- PHP-эндпоинты отдают `Content-Type: application/json; charset=utf-8`, доступны **без HTTP-авторизации**.
- Локальная проверка фронта без бэка — статические фикстуры в `public/data/`
  (`employees.example.json`, `tasks.example.json`), подставляются в `data-table-src` / `data-tasks-src`.
- Создание/редактирование/удаление сейчас demo (без бэка) — сохранение на сервер это отдельный шаг.
